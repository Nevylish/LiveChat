LiveChat est un bot Discord et un overlay OBS Studio/Streamlabs inspiré par la Cacabox qui permet d'afficher une image, une vidéo ou jouer un son sur un flux en direct depuis une simple commande Discord.

Démonstration

La vidéo a été enregistrée directement avec OBS Studio et la source LiveChat en tête de liste, ça donne un aperçu du côté spectateur. Attention, le son peut surprendre à 30 secondes.

1. Installation du bot Discord

Pour commencer, invitez le bot sur votre serveur privé Discord.
Vous pouvez en créer un pour cette utilisation uniquement.
C'est grâce à lui que vous pourrez faire afficher du contenu sur votre overlay.
Tapez /livechat dans un salon textuel pour afficher la commande.
Nous reviendrons en profondeur sur son fonctionnement dans la troisième partie.
Gardez à l'esprit que tous les membres présents sur le serveur pourront utiliser le LiveChat et faire apparaître n'importe quoi sur votre flux, n'invitez pas n'importe qui.
Ajouter le bot

2. Configuration de l'overlay

Pour obtenir votre lien, rentrez les informations demandées ci-dessous puis cliquez sur le bouton Générer mon lien d'overlay.

Maintenant que vous avez votre lien, vous allez ajouter une source Navigateur sur l'une de vos scènes, puis vous l'insérez dans la case URL.
Changez la largeur et la hauteur pour celle configurée dans les paramètres d'OBS, généralement 1920 de largeur et 1080 de hauteur.
Cochez Contrôler l'audio via OBS, cela est obligatoire pour les prochaines étapes.
Dans la partie CSS personnalisé, copiez et ajoutez le texte ci-dessous, cela permet de masquer la page d'erreur de Cloudflare si LiveChat est hors-ligne.
#cf-wrapper { display: none; }

Cliquez sur OK pour valider et placez la source au dessus de toutes les autres.
Pour activer le son de LiveChat: Faites un clic droit sur le mélangeur audio, allez dans les Paramètres audio avancées, trouvez la source de LiveChat et dans la catégorie Monitoring Audio sélectionnez Monitoring et sortie.
Vous pourrez maintenant diminuer ou augmenter le volume de LiveChat.
Vous pouvez désormais dupliquer la source sur l'ensemble des scènes que vous souhaitez.

3. Utiliser LiveChat

Utilisez la commande /livechat pour partager des médias, vous aurez plusieurs options à remplir:
  • Cible: Séléctionnez le streameur dans la liste
  • URL: Lien brut/direct vers le fichier1
  • Texte: Permet d'ajouter du texte sur le bas du média avec la police d'écriture Impact (style Meme)
  • Fullscreen: Affiche le média en plein écran

1 📁 Formats de médias supportés:

LiveChat supporte les liens directs vers des médias ainsi que les liens Tenor et X (anciennement Twitter), l'ajout d'autres plateformes est prévu.
Voici comment obtenir un lien direct:
  • Téléchargez le média sur votre pc (notube.lol pour YouTube par exemple)
  • Envoyez votre média dans un canal Discord, sur le serveur que vous avez créé pour LiveChat par exemple
  • Faites un clic droit sur le fichier et sélectionnez "Copier le lien"
Formats de fichier pris en charge: .mp4, .webm, .mkv, .mov, .mp3, .wav, .ogg, .jpg, .png, .gif
⚠️ Important: Les liens YouTube, TikTok ou autres plateformes de streaming non citées au dessus ne sont pas supportés. Vous devez d'abord télécharger le média et pour obtenir un lien direct.
