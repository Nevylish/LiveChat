Installation du bot Discord

Pour commencer, invitez le bot sur votre serveur privé Discord. Vous pouvez en créer un dédié à cette utilisation.

C'est grâce à lui que vous pourrez faire afficher du contenu sur votre overlay.

Tapez /livechat dans un salon textuel pour afficher la commande.
Nous reviendrons en profondeur sur son fonctionnement dans la quatrième partie.

Gardez à l'esprit que tous les membres présents sur le serveur pourront utiliser le LiveChat et faire apparaître du contenu sur votre flux. N'invitez pas n'importe qui.

Installation de l'overlay

Suivez ces étapes pour ajouter l'overlay à OBS Studio ou Streamlabs.
Pour les autres logiciels référez-vous à leur documentation pour ajouter une source Navigateur.

  1. Ajoutez une source Navigateur
  2. Collez le lien généré précédemment dans URL
  3. Réglez la taille sur 1920x1080
  4. Cochez Contrôler l'audio via OBS
  5. Cliquez sur OK et placez la source au-dessus
  6. Dans le mélangeur audio: clic droit > Propriétés audio avancées > LiveChat > Monitoring et sortie

Utiliser LiveChat

Utilisez la commande /livechat sur Discord pour partager des médias.

  • Cible : Sélectionnez le streamer
  • URL : Lien direct ou TikTok, X, Giphy, Tenor, Discord
  • Fichier : Upload direct depuis votre PC
  • Texte : Ajoute un texte style Meme
  • Fullscreen : Affiche le média en plein écran

Formats de fichier pris en charge: .mp4, .webm, .mkv, .mov, .mp3, .wav, .ogg, .jpg, .png, .gif

Important: Les liens YouTube ou autres plateformes de streaming non cités ne sont pas supportés directement. Téléchargez le média d'abord.
